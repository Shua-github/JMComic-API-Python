# 还在施工(虽然一部分可以用了)

## 运行
```
git clone https://github.com/Shua-github/JMComic-API-Python.git #拉取
cd JMComic-API-Python #进入文件夹
pip install -r requirements.txt #安装依赖
python app.py #启动
```

## 使用
访问`IP:端口`查看`Fastapi`自带的文档(默认端口是`5000`)
